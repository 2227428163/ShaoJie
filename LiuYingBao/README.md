# 烧结生产工艺
传统的带式抽风烧结机是目前最常用的烧结设备。本文以某大型联合钢铁企业的 360 平方米（90 米长，4 米宽）带式抽风烧结机作为研究对象，它有 24 个抽风风箱，其生产过程如图所示。烧结生产工艺包括配料、混合制粒、布料、点火、抽风烧结、破碎筛分等工序。
